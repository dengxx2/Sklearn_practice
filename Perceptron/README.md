# Perceptron.ipynb 程序说明

使用感知机作为二分类模型

1. 原始数据划分成训练集和测试集

> (1). 打乱数据
> (2). 20%留做测试集,80%作为训练集和验证集,分别存储下来

2. 读取训练集数据,并划分为80%的训练集,20%的验证集
3. 特征标准化
4. 训练感知机模型
5. 5-fold 交叉验证
6. 存储和加载模型
7. 测试集上的准确度
